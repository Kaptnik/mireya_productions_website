# Mireya Productions Website

A modern, artistic website for Mireya Productions movie production company featuring a dark theme, elegant typography, and responsive design.

## About Mireya Productions

Founded in 2025 by visionary filmmaker Peony Patnaik, Mireya Productions is an emerging force in the film industry dedicated to creating transformative cinematic experiences. The company focuses on developing meaningful projects that explore important themes through innovative storytelling.

## Features

- **Dark, Cinematic Theme**: Professional dark color scheme with gold accents
- **Artsy Typography**: Uses elegant fonts (Playfair Display & Cinzel) for a sophisticated look
- **Responsive Design**: Fully responsive across all devices
- **Smooth Animations**: CSS animations and transitions for enhanced user experience
- **Interactive Navigation**: Mobile-friendly navigation with smooth scrolling
- **Portfolio Filtering**: Dynamic filtering on the Work page for upcoming projects
- **Modern Layout**: Grid-based layouts with clean, professional aesthetics

## Pages

1. **Home** (`index.html`) - Hero section with upcoming projects and company overview
2. **Work** (`work.html`) - Portfolio showcase with placeholder projects in development
3. **Producers** (`producers.html`) - Founder profile and company vision
4. **About Us** (`about.html`) - Company story, mission, values, and timeline

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Cinzel)

## Design Features

- **Color Palette**: Dark backgrounds (#0a0a0a, #1a1a1a) with gold accents (#d4af37, #f4e79d)
- **Typography**: Elegant serif fonts for headings, clean sans-serif for body text
- **Layout**: Modern grid-based layouts with card designs
- **Animations**: Smooth hover effects, fade-in animations, and page transitions
- **Interactive Elements**: Filtering buttons, mobile navigation, smooth scrolling

## Getting Started

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Navigate through the different pages using the navigation menu

## Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

## Customization

The website is built with modular CSS that makes it easy to customize:

- Colors can be adjusted in the CSS variables section
- Typography can be changed by updating the Google Fonts imports
- Layout modifications can be made through the CSS Grid and Flexbox properties
- Content can be easily updated in the HTML files
